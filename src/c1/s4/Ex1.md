1.4.1 证明从N个数中取三个整数的不同组合总数为N(N-1)(N-2)/6，使用数学归纳法

```text
令 C(N) = N(N-1)(N-2)/6

N=1     C(N) = 0
N=2     C(N) = 0
N=3     C(N) = 3(3-1)(3-2) / 6 = 1

假设 N=M 时，C(M) = M(M-1)(M-2) / 6 成立

证明 N=M+1时，C(M+1) = (M+1)M(M-1) / 6

令C2(N)为从N个数中取两个整数的不同组合总数
C2(N) = N(N-1) / 2

C(M+1) = C(M) + C2(M)
       = C(M) + M(M-1) / 2
       = M(M-1)(M-2) / 6 + 3M(M-1) / 6
       = M(M-1)(M-2+3) / 6
       = M(M-1)(M+3) / 6
       
得证
```
